# Regras do Projeto

* Nunca faça nenhuma operação no Git (add, commit, push, merge, pull, reset, checkout, etc.) sem que o usuário peça explicitamente.
* Jamais, sob hipótese alguma, coloque na aplicação (ou preencha por meio da IA) informações/dados que não tenham sido de fato retirados do arquivo PDF anexado ao respectivo processo.
